# SC360 Supply chain Intelligence v0.8.0 - No Install

## Start on Windows
Double-click `START_SC360_WINDOWS.bat`.

Then open the address shown in the command window (normally http://localhost:4173).

## Modes
1. **Demo Factory** - five planners, five daily snapshots, change detection and Scenario Studio.
2. **Live Expedite Upload** - preserved v0.6.2 upload workflow for `.xls` and `.xlsx` reports.

## Source
The TypeScript source for the next compiled release remains in `src/`. The self-contained v0.7 Demo Factory runtime is in `dist/` and does not require package installation.


## v0.8.0 Decision Workflow

Open a constraint and select **Review recovery options**. SC360 ranks operational solutions, presents cost and impact, and allows the planner to escalate a complete proposal into the **Decision Center** for management approval.
