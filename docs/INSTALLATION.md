# Installation and Startup

## Recommended Windows startup

No dependency installation is required.

1. Extract the complete ZIP.
2. Double-click `START_SC360_WINDOWS.bat`.
3. Keep the terminal window open.
4. Use SC360 at `http://localhost:4173`.

Requirements: Node.js LTS only.

## Development startup

Double-click `START_DEVELOPMENT_WINDOWS.bat`, or run:

```bash
npm install
npm run dev
```

All npm configuration in this release targets the public registry.
