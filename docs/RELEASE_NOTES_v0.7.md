# SC360 Expedite Intelligence v0.7.0

## Demo Factory
- Five planner workspaces with separate supplier portfolios.
- Five sequential Expedite snapshots (Monday-Friday).
- Factory and planner-specific Control Rooms.
- Daily change intelligence: new, worsening, improving, stable and resolved.
- Planner workload and factory portfolio comparison.
- Scenario Studio for supplier delay, demand spike, ASN receipt, quality hold, transport delay and capacity loss.
- Existing live Expedite upload application preserved under Live Expedite Upload.

## Data compatibility
The demo snapshots retain the generated Expedite format with Buyer, Supplier, Part Number, Production Cell, Past Due, weekly demand, purchase-order commitment, ASN and supplier-message fields.
